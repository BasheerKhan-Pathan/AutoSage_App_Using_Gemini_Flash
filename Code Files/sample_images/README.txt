SAMPLE VEHICLE IMAGES
=====================

This folder is for storing sample vehicle images to test the AutoSage app.

WHERE TO FIND VEHICLE IMAGES:
------------------------------
1. Pexels (Free stock photos): https://www.pexels.com/search/car/
2. Unsplash (Free images): https://unsplash.com/s/photos/vehicle
3. Pixabay (Free images): https://pixabay.com/images/search/automobile/
4. Your own vehicle photos
5. Manufacturer websites (for reference only)

TIPS FOR BEST RESULTS:
----------------------
- Use clear, high-resolution images (at least 800x600 pixels)
- Front or side view works best
- Good lighting is essential
- Avoid blurry or heavily filtered images
- Close-up shots of the full vehicle work better than distant shots

SUPPORTED FORMATS:
------------------
- JPG / JPEG
- PNG

EXAMPLE SEARCHES:
-----------------
- "Honda motorcycle"
- "Tesla Model 3"
- "Yamaha R15"
- "Toyota Camry"
- "BMW M3"
- "Harley Davidson"

Simply download images from the above sources and save them here for easy access!

NOTE: Respect copyright and usage rights when downloading images.
